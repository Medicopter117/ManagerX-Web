📚 Einführung

Willkommen zur Dokumentation von ManagerX. Hier findest du eine Übersicht über alle Befehle, Kategorien und Funktionen mit Beispielen.

🛠️ Moderation

  • /kick [user] – Entfernt ein Mitglied vom Server
  • /ban [user] – Bannt ein Mitglied dauerhaft
  • /mute [user] [dauer] – Sperrt den Nutzer zeitweise
  • /warn [user] [grund] – Erteilt eine Verwarnung
  • /warnings [user] – Zeigt alle Verwarnungen

🎉 Fun

  • /joke – Zeigt einen zufälligen Witz
  • /meme – Holt ein Meme aus dem Internet
  • /8ball [Frage] – Magische Kugel mit Antwort

📊 Statistik

  • /stats – Zeigt deine Server-Aktivitäten
  • /leaderboard – Bestenliste der aktivsten Nutzer

📁 Notizen

  • /note add [text] – Neue Notiz erstellen
  • /note view – Eigene Notizen anzeigen
  • /note delete [ID] – Notiz löschen

🔊 TempVC

  • /tempvc create – Erstellt temporären Sprachkanal
  • /tempvc settings – Optionen für den TempVC

🌤️ Wetter

  • /weather [Stadt] – Wetter für gewünschte Stadt

👤 Userinfo

  • /userinfo [user] – Zeigt Infos über den Nutzer
